  1. Select Species: Choose Dog or Cat (dosing differs significantly)
  2. Enter Weight: Input patient weight in kg, lb, or grams using the unit toggle
  3. Select Drug: Choose from the dropdown or type to search
  4. Select Form: Oral (tablet/liquid) or Injectable (IV/IM/SC)
  5. Enter Concentration: Input drug concentration from vial/label (e.g., 50 mg/mL)
  6. Review Results: Check calculated dose, volume, and safety warnings
